{
  "short": "30 字以内的中文一句话标题，要醒目、信息密度高",
  "card": "100-200 字中文卡片摘要，说清项目做什么、技术栈、关键卖点",
  "problem": [],
  "audience": [],
  "why_trending": []
}

强制规则：
1. 只基于 README + 元数据，禁止编造未提及的事实（如版本号、发布事件、对比其他项目）

3. card 100-200 字、不要用"这是一个"开头、不要用"非常"等廉价副词
4. problem / audience / why_trending 数组在本版本保持为空数组 []
5. 输出必须是合法 JSON，无 markdown 代码块包裹
